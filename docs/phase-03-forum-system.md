# 阶段 3：校园社区论坛系统

## 分支

`phase/03-forum-system`

## 目标

完成类似小红书体验的校园社区，支持图文发布、Tag、瀑布流、互动、热榜、词云和 AI 热点总结展示。

## 交付内容

- 图文帖子发布、敏感词校验、最多 5 个 Tag、单 Tag 最长 20 字符。
- 社区首页瀑布流布局。
- 帖子搜索、Tag 筛选、作者筛选、最新/热门排序。
- 一级评论、二级回复、评论点赞、@ 用户。
- 帖子点赞、收藏、分享、关注作者。
- 个人主页展示用户发布内容。
- 日榜、周榜、总榜热度计算。
- 每周词云数据和前后台展示。
- AI 社区热点总结展示与管理员重新生成入口。
- 帖子/评论举报和管理员处理。

## 用户体验补充

- 发布器提供 Tag 数量和长度实时提示。
- 瀑布流在无图内容下仍保持稳定卡片比例。
- 互动后即时反馈并同步通知中心。
- 热榜解释“为什么上榜”，避免只显示冷冰冰分数。
- 举报入口不打断阅读，提交后提供处理状态追踪。

## 验收点

- 用户可发布图文帖子。
- 社区首页为瀑布流布局。
- Tag 规则和敏感词过滤生效。
- 帖子可点赞、评论、收藏、关注。
- 支持楼中楼回复、热榜、词云和 AI 总结展示。
- 管理员可处理社区举报。
