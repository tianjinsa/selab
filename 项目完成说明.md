# 校园智能生活服务平台完成说明

## 项目结构

- `weixin`：微信小程序端，基于 TDesign 小程序组件，包含任务互助、二手市场、社区论坛、智能体、我的五个主 tab。
- `web`：管理员后台，Vue3 + Element Plus，构建产物输出到 `server/public`。
- `server`：Node.js + Express 后端，包含统一认证、消息通知、任务、社区、二手、智能体、后台管理 API，以及 WebSocket 私信入口。

## 启动方式

后端服务：

```bash
cd server
npm install
npm start
```

管理员后台开发预览：

```bash
cd web
npm install
npm run dev
```

管理员后台构建到后端静态目录：

```bash
cd web
npm run build
```

微信小程序：

```bash
cd weixin
npm install
npm run lint
```

然后使用微信开发者工具导入 `weixin` 目录并构建 npm。

## SQL Server 初始化

业务数据库名为 `CampusSmartLifeDB`，初始化连接上下文使用 `tempdb`。

```bash
cd server
npm run db:init
```

该命令会删除旧的 `CampusSmartLifeDB` 并重新创建数据库，然后执行 `server/sql/schema.sql` 建表。

只检查 SQL Server 连接：

```bash
cd server
npm run check:sqlserver
```

当前后端默认使用 `server/data/db.json` 演示数据仓库，保证在 SQL Server 凭据不可用时仍可运行完整演示。
